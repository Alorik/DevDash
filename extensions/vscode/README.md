# DevsDash VS Code Extension

Track your coding activity directly from VS Code and sync it with DevsDash.

## Features

- ⏱️ Track coding time
- 📁 Detect active files and languages
- 🔄 Send activity to DevsDash API
- 📊 Power your analytics dashboard

## How it works

The extension listens to editor events and sends activity data to your backend API.

## Setup

1. Install the extension
2. Start coding
3. View analytics in DevsDash
